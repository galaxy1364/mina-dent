LOCKPACK RULES (User Non-Negotiables)
- Default: STOP تا وقتی LOCKPACK کامل نشده.
- PASS فقط با: 0 JS runtime errors (onerror/unhandledrejection/console.error=0) + No infinite pending (timeouts قطعی) + Abort-safe (ABORTED) + Signed CI artifact + Evidence کامل.
- هیچ bypass/استثناء حتی برای admin.
- تغییرات فقط در 3 مسیر: Proceed / Fix-to-PASS / Rollback. Scope Freeze تا PASS فاز جاری.
- بدون Evidence = انجام نشده.
- گیت‌ها ثابت و Required.
